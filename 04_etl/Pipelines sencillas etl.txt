Crear tabla temp_dim_tiempo en BigQuery
SQL:
CREATE OR REPLACE TABLE `cadena-de-supermecados.stg_supermercado.temp_dim_tiempo` AS
SELECT
  ROW_NUMBER() OVER (ORDER BY fecha) AS id_tiempo,
  fecha,
  EXTRACT(DAY FROM fecha) AS dia,
  EXTRACT(MONTH FROM fecha) AS mes,
  CASE EXTRACT(MONTH FROM fecha)
    WHEN 1 THEN 'enero' WHEN 2 THEN 'febrero' WHEN 3 THEN 'marzo'
    WHEN 4 THEN 'abril' WHEN 5 THEN 'mayo' WHEN 6 THEN 'junio'
    WHEN 7 THEN 'julio' WHEN 8 THEN 'agosto' WHEN 9 THEN 'septiembre'
    WHEN 10 THEN 'octubre' WHEN 11 THEN 'noviembre' ELSE 'diciembre'
  END AS nombre_mes,
  CONCAT('Q', CAST(EXTRACT(QUARTER FROM fecha) AS STRING)) AS trimestre,
  EXTRACT(YEAR FROM fecha) AS anio,
  CASE EXTRACT(DAYOFWEEK FROM fecha)
    WHEN 1 THEN 'domingo' WHEN 2 THEN 'lunes' WHEN 3 THEN 'martes'
    WHEN 4 THEN 'miercoles' WHEN 5 THEN 'jueves' WHEN 6 THEN 'viernes'
    ELSE 'sabado'
  END AS dia_semana,
  CAST(NULL AS STRING) AS franja_horaria
FROM UNNEST(GENERATE_ARRAY(0, DATE_DIFF(DATE '2026-12-31', DATE '2023-01-01', DAY))) AS n
CROSS JOIN UNNEST([DATE_ADD(DATE '2023-01-01', INTERVAL n DAY)]) AS fecha;


PIPELINE 1: pipe_dim_tiempo
Arquitectura: BigQuery Source → Wrangler → BigQuery Sink
Source BigQuery:
Dataset: stg_supermercado
Table: temp_dim_tiempo
Wrangler — directivas:
trim :nombre_mes
trim :dia_semana
Sink BigQuery:
Dataset: dw_supermercado
Table: dim_tiempo
Truncate: true
Output Schema nullable: solo franja_horaria
Verificación:
SQL:
SELECT COUNT(*) FROM `cadena-de-supermecados.dw_supermercado.dim_tiempo`;
-- Esperado: 1,461

PIPELINE 4: pipe_dim_promocion
Arquitectura: BigQuery Source → BigQuery Sink (sin Wrangler)
Paso previo — crear tabla staging en BigQuery:
SQL:
CREATE OR REPLACE TABLE `cadena-de-supermecados.stg_supermercado.stg_dim_promocion` AS
SELECT
  ROW_NUMBER() OVER (ORDER BY id_promocion) AS id_promocion_sk,
  id_promocion AS id_promocion_nk,
  tipo_promocion,
  TRIM(descripcion) AS descripcion,
  CAST(fecha_inicio AS STRING) AS fecha_inicio,
  CAST(fecha_fin AS STRING) AS fecha_fin
FROM `cadena-de-supermecados.stg_supermercado.tb_promocion`;
Crear tabla destino en BigQuery:
SQL:
CREATE OR REPLACE TABLE `cadena-de-supermercados.dw_supermercado.dim_promocion` (
  id_promocion_sk  INT64,
  id_promocion_nk  INT64,
  tipo_promocion   STRING NOT NULL,
  descripcion      STRING NOT NULL,
  fecha_inicio     STRING,
  fecha_fin        STRING

SQL post-carga:
SQL:
CREATE OR REPLACE TABLE `cadena-de-supermecados.dw_supermercado.dim_promocion` AS
SELECT
  d.id_promocion_sk,
  d.id_promocion_nk,
  d.tipo_promocion,
  d.descripcion,
  SAFE.PARSE_DATE('%Y-%m-%d', d.fecha_inicio) AS fecha_inicio,
  SAFE.PARSE_DATE('%Y-%m-%d', d.fecha_fin)    AS fecha_fin,
  CAST(p.porcentaje_descuento AS NUMERIC)     AS porcentaje_descuento
FROM `cadena-de-supermecados.dw_supermercado.dim_promocion` d
LEFT JOIN `cadena-de-supermecados.stg_supermercado.tb_promocion` p
  ON d.id_promocion_nk = p.id_promocion;

SQL:
